# Proxym Device Reservation Project

## Description
Application web pour gérer la réservation de devices de test au sein de l’entreprise Proxym.  
Backend : NestJS + PostgreSQL  
Frontend : React  

---

## Prérequis
- Node.js (v18+ recommandé)  
- npm (v9+)  
- PostgreSQL (v15+)  
- pgAdmin (optionnel, pour gérer la base)  

---

## Backend (NestJS)

### 1. Installer les dépendances
```bash
cd prog
npm install
npm run start:dev

```

## Frontend (ReactJS)
```bash
cd reactprog
npm install
npm start
